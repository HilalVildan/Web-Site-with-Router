# Web Design wiht Router

Responsive web design site made using React

<hr />

## What I Used

- Bootstrap
- React Router Dom


### Installation Options

```
$ git clone https://github.com/HilalVildan/Web-Site-with-Router.git
```

```
$ npm install
```

```
$ npm start
```

### Or

```
$ git clone https://github.com/HilalVildan/Web-Site-with-Router.git
```

```
$ yarn
```

```
$ yarn start
```
